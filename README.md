# 📝 PyEditor Advanced

A simple Python Tkinter-based GUI text editor with useful features like syntax highlighting, tabs, auto-save, and more.

## 🚀 Features
- Multiple tabs
- Line numbers
- Syntax highlighting (Python)
- Auto indentation
- Dark / Light mode
- Find text
- Auto backup system

## 🛠️ Built With
- Python
- Tkinter

## ▶️ How to Run
1. Install Python
2. Run:python py_texteditor.py
   ## 💻 EXE Version
No need to install Python. Just run the .exe file.
