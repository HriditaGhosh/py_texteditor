# 📝 PyEditor Advanced

A simple Python Tkinter-based text editor with useful features like syntax highlighting, tabs, auto-save, and more.

## 🚀 Features
- Multiple tabs
- Line numbers
- Syntax highlighting (Python)
- Auto indentation
- Dark / Light mode
- Find text
- Auto backup system

## 🛠️ Built With
- Python
- Tkinter
